## Overview

The V1 ESCC was manufactured from 2019 to 2020. It is smaller in diameter than the V2 ESCC, and thicker than the OG/interim ESCCs.

## TCR

The TCR of your V1 ESCC can be found using the calculator below. Please note that this calculator _only_ works with V1 ESCCs.
